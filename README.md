# Discord Welcome Bot 🤖

A sleek Discord bot built with `discord.py` and `easy-pil` that generates dynamic welcome cards for new members and sends automated rules via DM.

## ✨ Features
* **Custom Welcome Cards:** Automatically generates an image with the user's avatar and name.
* **Auto-DM Rules:** Sends a stylish Embed to new members with server guidelines.
* **Status Tracking:** Displays a "Watching the server" status.
* **Secure Configuration:** Uses environment variables (`.env`) to protect your bot token.

## 🚀 Getting Started

* Python 3.8 or higher
